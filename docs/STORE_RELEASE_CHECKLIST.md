# Chrome Web Store release checklist

- [ ] Independent name, icon, screenshots, and listing copy.
- [ ] No analytics, remote code, external API, advertising, or telemetry.
- [ ] Permissions limited to storage, tabs, tabGroups, and contextMenus.
- [ ] Privacy policy published and linked.
- [ ] Single-purpose description explains rule-based tab grouping.
- [ ] Stable and Beta manual regression complete.
- [ ] Split View regression complete on a supported Chrome release.
- [ ] Browser-agent external-group regression complete.
- [ ] Release ZIP generated from a clean tagged commit.
- [ ] Source and store manifest versions match.
